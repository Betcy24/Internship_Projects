# Internship Projects – SMEC Labs, Kochi

This repository contains the projects and practical exercises completed during my internship at SMEC Labs, Kochi from August 2025 to March 2026. The internship provided hands-on experience in programming and software development, particularly focusing on Python and its practical applications.

Throughout this internship, I worked on multiple small projects and coding exercises that helped strengthen my understanding of Python fundamentals, including control statements, loops, functions, and basic project development. These projects demonstrate the foundational concepts of Python programming and are intended to help beginners learn and practice core programming skills.

The work completed during this internship significantly improved my technical knowledge, problem-solving ability, and understanding of real-world programming workflows. The projects in this repository can serve as a useful reference for anyone who wants to learn the basics of Python and understand how simple programs are structured and developed.

**Internship Details**
- Organization: SMEC Labs

- Duration: August 2025 – March 2026
